# Installation

## Windows
* [`Download Node JS`](https://nodejs.org/en/download/)
* [`Download Git`](https://git-scm.com/download/win)

## Cloning this repo
```cmd
> git clone https://github.com/Stalker-moment/yt-now-playing/
> cd yt-now-playing
```

## Install the package
```cmd
> npm i
```

## Run the script
```cmd
> npm start
```
